# 网易云音乐云盘管理
构建自己的音乐小库

## 本地开发
#### 启动文件上传服务
```
yarn serve
```
#### 启动界面
```
yarn dev
```

## 功能

- [x] 扫码登录、手机号登陆
- [x] 云盘上传
